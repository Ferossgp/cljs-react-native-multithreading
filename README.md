### Multithreading

Install deps:
```sh
$ yarn && cd react-native && yarn && npx pod-install
```

Build the Worker CLJS bundle:
```sh
$ yarn build-worker:cljs
```

Start the cljs app:
```sh
$ yarn start:app-cljs
```

Start iOS app:
```sh
$ yarn ios
```

Start metro bundler:
```sh
$ yarn start:metro
```
